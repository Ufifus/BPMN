# BPMN

## L1. Архитектура модуля телемедицинских консультаций для МИС

Файлы схем:

- BPMN 2.0: [`docs/telemedicine-consultation-l1.bpmn`](docs/telemedicine-consultation-l1.bpmn);
- Structurizr DSL / C4: [`docs/telemedicine-consultation-l1.dsl`](docs/telemedicine-consultation-l1.dsl).

Диаграмма описывает верхнеуровневый процесс телемедицинской консультации в МИС:

- подача заявки пациентом и проверка согласий/права на услугу;
- создание телемедицинского случая и подготовка сеанса;
- интеграции с расписанием, ЭМК, регистратурой, уведомлениями и видеоплатформой;
- проведение консультации врачом;
- публикация заключения, закрытие случая и обновление статусов оплаты/ОМС.

Версия BPMN оформлена как BPMN 2.0 XML и может быть открыта в редакторах, поддерживающих BPMN 2.0.

Версия Structurizr DSL описывает ту же предметную область как C4-модель:

- L1 System Context для модуля телемедицинских консультаций;
- Container view с личным кабинетом пациента, АРМ врача, API/оркестратором, адаптерами, БД и аудитом;
- внешние и смежные системы: ядро МИС, ЭМК, оплата/ОМС, видеоплатформа и шлюз уведомлений.
